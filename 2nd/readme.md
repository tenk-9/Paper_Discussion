### 論文を読むに当たって
- 被引用数：どれだけ読まれているか，参照されているか
  - 多いほどいい
- 著者と機関
  - 著者はコントリビューション順
  - 一番最後が責任者
  - 話者照合はフランス（ヨーロッパ）が強いので，そこの機関が絡んでいるのが良い．
- 発行機関
  - Arxivは査読が無いので，有象無象
    - `Journal reference`に，どこに投稿されたかが記載されている
  - 論文を検索して，Arxiv以外に掲載されていたら，査読が通ったか否かがわかる
